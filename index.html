阶层、现状以及对未来的思考

By linboz

发表于 2016-03-06

最近北上深的房价直线飙升，尤其是深圳，从2015年开始至今日，房价一路高歌，屌丝如我只能望房兴叹。有房者沾沾自喜，以一副既得利益者的姿态，在各种社交媒体上傲然自得。
确实，因为中国特殊的环境，最优质的资源，如教育、医疗、就业、生活等资源都集中在北上深广等几个大城市。人要生存，就得解决住房问题，而城市中的土地有限，各种需求又庞大，根据经济学上简单的供求关系，房价的上涨长期看是必然。
因为中国的房价问题，可以说已经拉开了穷人与中产、富人阶层的差距，屌丝买不起房，只能打工勉强维持一家老小生存，根本没有时间和闲钱去规划自己的投资。富人占据了一线城市的优质资源，享受房价飙升、资产增值的过程中，又在不断地寻找投资机会，于是强者恒强，穷者恒穷，阶层固化由此形成。
我们常常以为，只要努力就有回报。现实的高房价彻底击碎了这种天真的妄想。你打工一个月，月入过万又怎样？还是买不起大城市房子的一平米——光是应付一家老小的生活费用就够呛了，还想着跨入有房阶层？
深圳，这个中国未来最有发展潜力的移民城市，高房价在一方面正是说明未来的光明前景。股市上有一个名词叫市盈率，用在深圳现在最适合不过。高市盈率，意味着深圳未来的成长性足够、空间很大。
目前的现状，总的来说归结为四个问题：

  1. 工资低
  2. 资金少
  3. 技术能力不足
  4. 投资经验不足

无产阶级要想迈进中产阶级乃至富人阶级，首先必须要有钱，其次是对钱的把控能力（知道怎样用好钱，使其产生最大化的经济价值）。
没钱怎么办？
存钱。保持节俭有度的生活方式，用好每一笔钱。工资太低也存不了多少钱，因而要提高储蓄率，也就是工资要能够大幅提升。工资的提升，需要自身的工作能力、技术方面的提高，于是问题便成了如何提高自身的技术能力。当然，创业也是一个好路子。但我始终认为，创业是可遇不可求的，那些创业有大成者也许都是幸存者，并不具备统计意义。目前，最有把握的、最能够做好的事情，不去做好而去干那风险大、存活率低的工作，这不是一个理性的人所采取的最佳路径。
存钱，是为了投资。普通人的致富路子，其中之一就是进行投资。为了不踩坑，把辛辛苦苦赚的钱赔进去，学习一点投资的方法是必须滴。我自己列了一个书单，供以后反复阅读：

  • 聪明的投资者
  • 炒股的智慧
  • 漫步华尔街
  • 股票作手回忆录
  • 彼得林奇的成功投资
  • 怎样选择成长股
  • 投资最重要的事
  • 穷查理宝典
  • 百箭穿杨
  • 孙子兵法

有些书，读完后要总结、实践才能掌握书中的奥义。后续我打算把以前读过的一些经典的书籍作一个归纳总结。

漫步华尔街——迈向稳步致富之路

By linboz

发表于 2016-02-29

很久没有写读书笔记了，总觉得不写点什么一味地读书容易脑子混乱，必须写下来总结才能形成自己的东西。

漫步华尔街

随机游走（random walk)是指基于过去的表现，无法预测将来的发展步骤和方向。这一术语用于股市上，则意味着股票价格的短期走势不可预知，意味着咨询服务/收益预测和复杂的图标模型全无用处。若将这术语的逻辑内涵推向极致，便意味着一只戴上眼罩的猴子，随意向报纸的金融版面掷一些飞镖，选出的投资组合就可以与投资专家精心挑选出的一样出色。

作者对华尔街的投资人士企图在短期内预测股市走势，并主动选择投资的做法相当蔑视，即使华尔街人士短期内取得了投资胜利，也只不过运气使然。

在我的观念中，投资是一种为了获得合理预期收入（股息，利息或租金）或长期价值增值而购买资产的方法。投资与投机的区别，就在于对投资回报期的定义和回报的可预测性。

为什么要投资呢？

因为通货膨胀，货币会贬值，现金如果不通过一些方法增值保值的话，许多年后将是一堆废纸。

我们必须采取一些投资策略来维持实际购买力不变。否则，我们的生活水平将沦入每况愈下的厄运境地。

估值方法

资产估值的方法有两种：磐石理论（firm-foundation theory）和空中楼阁理论（castle-in-the-air theory）

磐石理论

磐石理论认为，无论是普通股股票还是不动产，每一种投资工具都具有被称为“内在价值”的坚实支柱。“内在价值”可以通过对现状和未来前景的细致分析获得。当市场价格低于（或高于）内在价值磐石时，买入（或卖出）的机会就会出现，因为波动最终会被矫正——这便是该理论的核心所在。

那如何计算股票的真实内在价值呢？威廉在《投资价值理论》一书中指出，股票的真实价值等于该股票未来所有股息的现值（贴现值）。

贴现的含义其实就是逆看收入。举例来说，就是不要看过了一年你将有多少钱（存入银行一美元，利率为5%，过一年就有1.05美元），而是看未来的钱现在为多少（即一年后的一美元现在只值95美分；如果现在投资95美分，投资回报率为5%，那么一年后大约能取得一美元）。
所有的未来收入都不如到手的钱有价值，原因在于如果你现在持有资金，它会为你赚取利息。

公司的增长率越高/增长持续时间越长/股息派发越多/股票风险越低/利率平均水平越低，其证券的磐石价值（以及市盈率）就会越高。

磐石理论（或价值投资）的缺点？

磐石理论依赖一些对未来增长程度和期限的难以处理的预测，这便是其要害所在。

空中楼阁理论

不注重分析股票的内在价值，而去分析大众投资者未来可能的行为模式。成功的投资者总是事先判断出最易被大众建造成空中楼阁的投资形势，随后抢先购买股票。
典型人物事迹：凯恩斯每天早上只在床上操盘半个小时，挣了几百万英镑。

在这样一个世界里，每一分钟都会诞生一个笨蛋，他的存在就是以高于你投资支付的价格购买你的投资。只要有人肯买，任何价格都不算高。其中没有别的原因，就是大众心理。

庸众疯狂投机案例

  1. 荷兰郁金香球茎热
  2. 英国南海泡沫
  3. 华尔街一败涂地

现代投资组合理论

  1. 优点：降低风险，抹平个股的波动
  2. 实践：分散化（不同行业标的）/多样化投资（国内国际市场兼顾）

资本资产定价模型

  1. 逻辑：能被多样化消除的风险不可能产生任何溢价，即多样化的投资组合不能提高投资收益率。
  2. 工具：β，系统性风险，对市场变动的相对波动性或敏感性。大于1时，波动变大，收益也可能变大。
  3. 缺陷：高β值的投资组合不一定能提供比低β值的投资组合更好的长期收益，仅凭β不能概括一只证券的全部系统性风险。
  4. 结论：β值与收益率之间没有长期的相关关系，但β仍然能够作为一个有用的投资管理工具，如存在这样一个事实，低β值股票和高β值股票有一样收益率，选择低β值股票

决定股票和债券收益的因素

普通股的长期回报率取决于两个至关重要的因素：购买时的股息率与未来的收益和股息增长率。原则上，对于准备永久持股的购买者来说，普通股的价值等于未来股息流的“现值”或“贴现值”。
长期股票收益率 = 初始股息率 + 增长率
债券获得的收益大约等于购买时已经确定的到期收益率。利率上升，债券价格下跌，利率下降，债券价格就会上升。

资产配置的四项原则

  1. 历史证明风险与收益总是结伴而行。
  2. 投资普通股和债券的风险取决于投资持有期限。持有期越长，风险越低。
  3. “定期等额投资法”虽然颇有争议，但却是一种降低股票和债券投资风险的有用方法。
  4. 你必须要分清楚，自己对待风险的态度和风险的能力是不同的。你所能承受风险的大小取决于你的整体财务状况，包括除投资以外你的收入类型和来源。

作者最后在给散户的建议中很推崇指数化投资的策略，认为此种方法适合绝大多数人，费用低廉，能够取得市场的平均回报，战胜大多数主动型基金经理。

若是自己非要进行选股操作，作者给出了一些原则作为参考：

原则1：限定选股范围，所选公司要能至少连续五年保持高于平均水平的收益增长率。
原则2：永远不要比一只股票合理的磐石价值多付一分钱。寻找尚未被市场发掘，市盈率还有巨大上升空间的成长前景。
原则3：市场上总会流传关于某些公司未来业绩增长的传闻，如果投资者愿意并且能够借此建造“空中楼阁”，那么购买这些股票将有利可图。获取成功的关键在于，你要先知先觉，抢占先机，你要提前几个月就抵达其他投资者将会蜂拥而至的地方。
原则4：尽可能少交易。

麦基尔法

购买高折价的封闭式基金，能够获取丰厚的回报。原因在于：

如果你购买了折价率为25%的基金，那么你每投资三美元，就可获得价值为四美元的资产所分配的红利。所以，就算基金的回报率仅仅和大盘相等，你还是可以如“市场漫步信奉者”所期待的那样，获得一个超过平均水平的收益。
这就好比你拥有一个利率为5%的100美元的储蓄账户，你存入100美元，每年获得五美元的利息。假如现在这个账户以25%的折价率出售，换句话说，你只需75美元便可买到这个账户，你每年照样获得五美元的利息，但是因为你只为这个账户支付75%美元，所以你的实际收益率是6.67%（5÷75）。
